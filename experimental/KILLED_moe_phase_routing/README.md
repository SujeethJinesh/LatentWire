# KILLED: MoE Phase Routing

## What Was Tried

MoE phase routing would route reasoning phases to different experts or cache
policies.

## Why It Died

The routing literature is crowded and moving quickly. A competitive paper would
need deeper MoE routing baselines and model access than this sprint can support.

## Salvage Value

The idea can be revisited after the hybrid-quantization branches resolve, but
it should not consume current Mac or GPU time.
