\section{Introduction}

Residual Migration asks whether high-magnitude residual values can be clipped
or migrated without destroying downstream reasoning accuracy. The Phase 0
surface is intentionally narrow: it tests whether a severe residual clipping
ablation causes a measurable exact-match accuracy drop on 12 frozen AIME-2025
traces from \texttt{ibm-granite/granite-4.0-h-tiny}. The run directory is
\url{experimental/residual_migration/phase0/results/rm_phase0_20260508T161556Z},
with artifact completeness recorded in
\url{experimental/residual_migration/phase0/results/rm_phase0_20260508T161556Z/artifact_check.json}.

The current paper status is not COLM- or ICLR-ready. The checker PASS says
that this frozen packet did not show an accuracy drop under the preregistered
replicates rule. It does not show preserved capability, because the unablated
baseline answered zero of the 12 prompts correctly. That headroom failure is
the central limitation of this draft and is recorded in
\url{experimental/residual_migration/phase0/results/rm_phase0_20260508T161556Z/metrics.json}
and
\url{experimental/residual_migration/phase0/results/rm_phase0_20260508T161556Z/checker_result.json}.

\section{Interpretation}

The saturated Phase 0 fact is narrow: under this exact frozen packet, the
95th-percentile residual clipping ablation produced no measured accuracy drop.
The live scientific branch is not yet a positive method, because the baseline
also had no correct answers. A zero drop from 0.0 baseline accuracy cannot
distinguish harmless residual migration from an evaluation surface with no
headroom.

The highest-priority next branch is Phase 1 with headroom and oracle safeguards.
Before any residual-quantization recipe claim, the evaluation must include a
surface where the baseline succeeds often enough for a drop to be meaningful,
an oracle or diagnostic showing recoverable task headroom, paired uncertainty,
and a strict separation between same-family replication and cross-family
falsification.

\section{Limitations}

This packet should not be cited as evidence that residual clipping preserves
reasoning capability. The relevant artifact facts are baseline accuracy 0.0,
ablation accuracy 0.0, and accuracy drop 0.0 in
\url{experimental/residual_migration/phase0/results/rm_phase0_20260508T161556Z/metrics.json}
and
\url{experimental/residual_migration/phase0/results/rm_phase0_20260508T161556Z/checker_result.json}.
Because no baseline prompt was correct, the CI [0.0, 0.0] in
\url{experimental/residual_migration/phase0/results/rm_phase0_20260508T161556Z/bootstrap_ci.json}
is a replicate-pass diagnostic, not capability-preservation evidence.

The result is single-model, 12-trace, and Phase-0-only. It does not yet include
larger frozen slices, seed repeats beyond the recorded prompt bootstrap,
cross-family falsification, oracle/headroom diagnostics, latency measurements,
or a deployable compression method.
