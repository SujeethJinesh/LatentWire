## Current Read

- Strongest positive-method candidate: **strict route selection + target-side
  process repair** on held-out GSM70/SVAMP70.
- Strongest frozen transport clue remains **grouped subspace transport +
  rank-4 residual**, but it is now supporting blocker evidence rather than the
  main positive lane.
- Strongest no-extra-repair external bar: **`C2C`**.
- Strongest honest paper framing today: **positive repair-and-routing method
  candidate with explicit budget-matching caveats**, backed by blocker and
  telemetry evidence explaining why frozen transport alone saturates.
- Strongest new additive design clue: **shared sparse feature dictionary /
  crosscoder interface**, currently toy-only but clearly stronger than raw
  residual transport in the shared/private feature setup.
- Strongest new efficiency clue: **format-gated test-before-repair**, which
  preserves held-out repair accuracy while saving repair calls, but does not
  yet increase task accuracy.
- Evaluation standard for the next paper table is now fixed: paired CIs,
  sample-level correctness, repair-call savings, tokens/bytes/latency, and
  exact comparator budgets must ship with the result.
