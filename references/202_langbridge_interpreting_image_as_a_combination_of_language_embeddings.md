# LangBridge

- Title: LangBridge: Interpreting Image as a Combination of Language Embeddings
- Date: 2025-03-25
- Link: https://arxiv.org/abs/2503.19404

Why this is in the backlog:

- Multimodal bridge paper with a clean “project into a language-native basis”
  framing.
- Useful inspiration for a learned or semi-learned bridge that maps source KV
  into a target-native subspace instead of relying on a purely static transport.
- Relevant if the next positive-method attempt adds a tiny projector or
  low-rank adapter rather than another retrieval heuristic.
