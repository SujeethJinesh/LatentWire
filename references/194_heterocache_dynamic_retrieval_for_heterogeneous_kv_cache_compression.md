# HeteroCache: A Dynamic Retrieval Approach to Heterogeneous KV Cache Compression

- Link: https://arxiv.org/abs/2601.13684
- Why it matters here: focuses on heterogeneous head behavior and dynamic retrieval in KV compression, which is close to our `2 -> 8` head-mismatch regime.
- Most useful takeaway for `latent_bridge`: heterogeneous transport likely needs head-specific dynamic retrieval structure rather than a one-size-fits-all descriptor.
