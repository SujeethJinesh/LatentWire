# Expected Attention

- Date: 2025-10-01
- Link: https://arxiv.org/abs/2510.00636
- Why it matters here: gives a direct recent reference for query-conditioned attention estimation, which is closer to the next live transport/bridge lane than static averaged templates.
- Most useful takeaway: expected attention is a stronger object than raw mean templates for preserving live retrieval behavior under a transport map.
