# AttAnchor

- Title: AttAnchor: Guiding Cross-Modal Token Alignment in VLMs with Attention Anchors
- Date: 2025-09-27
- Link: https://arxiv.org/abs/2509.23109

Why this is in the backlog:

- Direct reference for attention-anchor style alignment rather than only latent
  regression.
- Useful here because the repeated blocker is that live query structure is not
  preserved by static transport, and attention anchors suggest a way to inject
  query-aware structure into a bridge or routing policy.
