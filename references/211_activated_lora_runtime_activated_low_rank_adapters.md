# Activated LoRA: Runtime-Activated Low-Rank Adapters

- Date: 2025-04-16
- Link: https://arxiv.org/abs/2504.12397
- Why it matters here: suggests a tiny low-rank bridge can stay mostly dormant and activate only when the live query needs correction.
- Most useful takeaway: a query-conditioned activation boundary may stabilize adapter-style repairs better than one always-on static bridge.
