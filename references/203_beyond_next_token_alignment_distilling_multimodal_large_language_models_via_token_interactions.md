# Beyond Next-Token Alignment

- Title: Beyond Next-Token Alignment: Distilling Multimodal Large Language Models via Token Interactions
- Date: 2026-02-10
- Link: https://arxiv.org/abs/2602.09483

Why this is in the backlog:

- Useful reference for moving from static next-token alignment to richer
  token-interaction alignment.
- Relevant to the repeated blocker where offline calibration fit looks good but
  held-out reasoning remains dead.
- Strong inspiration for a future cross-model bridge objective that preserves
  query-key or token-interaction structure instead of only matching cached
  states.
